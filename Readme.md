# Statistical Learning Algorithms From Scratch

A collection of machine learning and statistical learning algorithms implemented from first principles using Python and NumPy.

The purpose of this repository is to develop a deeper understanding of how machine learning algorithms work internally by implementing them without relying on high-level machine learning libraries such as Scikit-Learn. Each implementation focuses on the mathematical intuition, optimization process, and computational foundations behind the algorithm.

## Motivation

Machine learning libraries make it easy to train models, but they often hide the underlying mechanics. This repository is an attempt to bridge that gap by rebuilding core algorithms from the ground up.

The goal is not to create production-ready implementations, but rather to understand the mathematics, optimization techniques, and design decisions that make these algorithms work.

## Algorithms

### Supervised Learning

* Linear Regression
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Naive Bayes
* Decision Trees
* Random Forests
* Support Vector Machines (SVM)

### Unsupervised Learning

* K-Means Clustering
* Principal Component Analysis (PCA)
* Hierarchical Clustering
* Gaussian Mixture Models (GMM)



## Design Philosophy

Every implementation in this repository follows a simple principle:

1. Start with the mathematical formulation.
2. Translate the equations into code.
3. Build the algorithm using only fundamental numerical operations.
4. Validate the implementation on real datasets.
5. Compare results with established machine learning libraries whenever appropriate.

## Technologies

* Python
* NumPy
* Pandas
* Matplotlib

Machine learning frameworks are intentionally avoided during implementation to maintain focus on the underlying algorithms.
